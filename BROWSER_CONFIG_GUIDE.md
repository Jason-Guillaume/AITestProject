# 浏览器配置和无头模式功能说明

## 功能概述

现在系统已经支持：
1. ✅ **多浏览器选择**：Chrome、Firefox、Edge
2. ✅ **无头模式**：有头/无头模式切换
3. ✅ **并发配置**：支持设置并发数（前端已收集，后端已传递）

## 实现方式：环境变量注入

系统通过环境变量将浏览器配置传递给脚本：

| 环境变量 | 说明 | 可选值 | 默认值 |
|---------|------|--------|--------|
| BROWSER_TYPE | 浏览器类型 | chrome/firefox/edge | chrome |
| HEADLESS_MODE | 无头模式 | True/False | False |
| PARALLEL_COUNT | 并发数 | 1-10 | 1 |

## 使用步骤

### 1. 使用支持多浏览器的脚本
- 文件：login_multi_browser.py
- 位置：D:\AITestProduct\media\ui_scripts\2026\05\01\

### 2. 在前端选择配置
- 选择浏览器：Chrome / Firefox / Edge
- 选择模式：有头模式 / 无头模式
- 设置并发数：1-10

### 3. 执行脚本并查看日志
