# tests 测试目录

## 模块概述
存放项目的 API 测试和集成测试。

### api/test_defect.py

### api/test_execution.py

### api/test_project.py

### api/test_sys_ai_config.py

### api/test_testcase.py
- **作用**: 用例管理测试

### api/test_user.py
