# AI 测试平台项目解析文档

## 项目概述

这是一个基于 Django 的 AI 驱动的智能测试平台，集成了测试用例管理、自动化执行、性能测试、缺陷追踪和 AI 辅助测试生成等功能。

## 项目结构

```
AITestProduct/
├── AITestProduct/          # 项目配置目录
├── user/                   # 用户管理模块
├── project/                # 项目管理模块
├── testcase/               # 测试用例模块
├── execution/              # 测试执行模块
├── defect/                 # 缺陷管理模块
├── assistant/              # AI 助手模块
├── server_logs/            # 服务器日志模块
├── common/                 # 公共模块
├── media/                  # 媒体文件存储
├── frontend/               # 前端项目
└── tests/                  # 测试目录
```
