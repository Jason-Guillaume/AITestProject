# 脚本执行中心模块

## 功能概述

脚本执行中心是一个统一的自动化测试脚本管理与执行平台，支持多种测试框架和执行模式。

## 模块结构

### 1. ScriptHub.vue - 脚本中心门户
- 暗黑科技风格的导航页面
- 4 个主要模块入口：
  - **Web UI 自动化**：已实现，支持 Selenium/Playwright
  - **接口自动化 (API)**：开发中
  - **App 自动化**：开发中
  - **性能压测脚本**：开发中

### 2. WebUIWorkbench.vue - Web UI 自动化工作台
- 工程管理：导入、编辑、删除 Web UI 自动化工程
- 执行控制台：配置浏览器、执行模式、并发数
- 实时日志：显示脚本执行过程
- 结果统计：展示测试用例执行结果

## 路由配置

```javascript
// 脚本中心入口
/script-hub

// Web UI 自动化工作台
/script-hub/webui
```

## 使用方式

1. 从侧边栏点击"脚本执行中心"进入门户页
2. 选择"Web UI 自动化"模块
3. 在工作台中导入测试工程
4. 配置执行参数并开始执行
5. 查看实时日志和执行结果

## 技术特性

- Vue 3 Composition API
- TypeScript 类型支持
- Element Plus 组件库
- 暗黑科技风格 UI
- 响应式布局设计
- 悬浮动效与外发光特效
