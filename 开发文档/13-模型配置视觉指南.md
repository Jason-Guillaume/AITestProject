## Interactive States

### Input Focus
```css
Before:
  border: 1px solid rgba(100, 116, 139, 0.3)
  background: rgba(0, 0, 0, 0.25)

  border: 1px solid #06b6d4
  box-shadow: 0 0 0 3px rgba(6, 182, 212, 0.15)
  background: rgba(0, 0, 0, 0.35)

```


## Usage Example

### Basic Configuration
```javascript
// User fills in:
Base URL:    https://api.openai.com/v1
API Key:     sk-proj-abc123...
Model ID:    gpt-4o

// Clicks "测试连接" → Success message
// Clicks "保存配置" → Configuration saved
```

### Advanced Configuration
```javascript
// User expands "Advanced Configuration"
Context Length:  8192

// Adds custom headers:
Header 1:  X-Custom-Auth → Bearer token123
Header 2:  X-Request-ID → req-456

// Clicks "保存配置" → All settings saved
```
