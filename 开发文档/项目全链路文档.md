## 章节概览（根据 `全项目剖析文档.md`）

1. **项目整体结构与技术选型** – 说明后端 Django/DRF、前端 Vue3、任务调度 Celery 等技术栈。
2. **数据库模型概览** – 列出每个业务模块的核心模型及关键字段，统一说明软删、审计字段。
3. **后端 API 与交互** – 汇总各模块的主要 REST 接口，说明请求/响应字段及权限控制。
4. **前端实现细节** – 描述前端目录结构、状态管理、主要页面交互流程（以缺陷、用例、执行为例）。
5. **控制台与脚本** – 统一列出 `manage.py` 子命令、批处理脚本 (`start‑all.bat` 等) 以及项目自带的辅助脚本 (`scripts/`) 的使用方式。
6. **文档生成机制** – `generate_docs.py` 自动解析后端/前端源码生成技术概览文档的工作原理。
7. **常见问题与排查** – 软删过滤、JWT 失效、Celery 任务卡顿等典型场景的排查步骤。

> **后续**：可在对应模块文档中继续补充 **底层运行细节**（如 Model `save` 调用链、Celery 任务调度实现、前端组件内部实现），以及 **可视化展示**（页面 UI 截图、流程图）。
