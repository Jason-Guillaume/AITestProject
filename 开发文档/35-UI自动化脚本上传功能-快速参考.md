# UI自动化脚本上传功能 - 快速参考

## 模型字段

```python
UIScriptUpload:
  - id: 主键
  - name: 脚本名称
  - script_type: LINEAR | POM
  - file_path: 文件路径
  - git_repo_url: Git仓库URL（可选）
  - entry_point: 执行入口
  - workspace_path: 工作空间路径
  - created_at: 创建时间
  - updated_at: 更新时间
  - is_active: 是否启用
```

## API快速参考

### 创建脚本
```bash
POST /api/assistant/ui-scripts/
Content-Type: multipart/form-data

参数:
- name: 脚本名称
- script_type: LINEAR 或 POM
- file_path: 文件（.py 或 .zip）
- entry_point: 执行入口路径
- git_repo_url: Git仓库URL（可选）
```

### 获取列表
```bash
GET /api/assistant/ui-scripts/
```

### 获取详情
```bash
GET /api/assistant/ui-scripts/{id}/
```

### 更新脚本
```bash
PUT /api/assistant/ui-scripts/{id}/
```

### 删除脚本
```bash
DELETE /api/assistant/ui-scripts/{id}/
```

### 切换启用状态
```bash
POST /api/assistant/ui-scripts/{id}/toggle_active/
```

### 获取工作空间信息
```bash
GET /api/assistant/ui-scripts/{id}/workspace_info/
```

## 文件限制

- **支持格式**: .py, .zip
- **最大大小**: 50MB
- **存储路径**: media/ui_scripts/%Y/%m/%d/
- **工作空间**: media/workspaces/{id}/

## 验证规则

1. 文件扩展名必须是 .py 或 .zip
2. 文件大小不超过 50MB
3. entry_point 必须指向存在的 .py 文件
4. ZIP 文件必须是有效的压缩包

## 工作流程

### 单文件上传
1. 上传 .py 文件
2. 保存到 media/ui_scripts/
3. workspace_path 设置为文件所在目录
4. 验证 entry_point 存在

### ZIP包上传
1. 上传 .zip 文件
2. 保存到 media/ui_scripts/
3. 解压到 media/workspaces/{id}/
4. workspace_path 设置为解压目录
5. 验证 entry_point 存在

### 删除脚本
1. 触发 pre_delete 信号
2. 清理工作空间目录
3. 删除数据库记录
4. 删除上传的文件

## 错误处理

| 错误 | HTTP状态码 | 说明 |
|------|-----------|------|
| 文件过大 | 400 | 超过50MB限制 |
| 文件类型错误 | 400 | 不是.py或.zip |
| ZIP损坏 | 500 | ZIP文件无法解压 |
| 入口点不存在 | 500 | entry_point文件不存在 |
| 入口点非Python | 500 | entry_point不是.py文件 |

## 代码示例

### Python客户端
```python
import requests

url = "http://localhost:8000/api/assistant/ui-scripts/"
headers = {"Authorization": "Token YOUR_TOKEN"}

# 上传单文件
with open('test.py', 'rb') as f:
    files = {'file_path': f}
    data = {
        'name': '测试脚本',
        'script_type': 'LINEAR',
        'entry_point': 'test.py'
    }
    response = requests.post(url, headers=headers, data=data, files=files)
    print(response.json())
```

## 数据库查询

```sql
-- 查看所有脚本
SELECT * FROM ui_script_upload;

-- 查看启用的脚本
SELECT * FROM ui_script_upload WHERE is_active = 1;

-- 按类型统计
SELECT script_type, COUNT(*) 
FROM ui_script_upload 
GROUP BY script_type;

-- 查看最近上传
SELECT id, name, created_at 
FROM ui_script_upload 
ORDER BY created_at DESC 
LIMIT 10;
```
