## 🎯 API端点总览

### 脚本管理
- POST `/api/assistant/ui-scripts/` - 上传脚本
- GET `/api/assistant/ui-scripts/` - 获取脚本列表
- GET `/api/assistant/ui-scripts/{id}/` - 获取脚本详情
- POST `/api/assistant/ui-scripts/{id}/toggle_active/` - 切换启用状态
- GET `/api/assistant/ui-scripts/{id}/workspace_info/` - 获取工作空间信息

### 执行管理
- POST `/api/assistant/ui-script-executions/execute/` - 执行脚本
- GET `/api/assistant/ui-script-executions/` - 获取执行记录列表
- GET `/api/assistant/ui-script-executions/{id}/logs/` - 获取执行日志
- GET `/api/assistant/ui-script-executions/{id}/status_detail/` - 获取执行状态详情

## 🚀 部署步骤

1. ✅ 安装依赖: `pip install redis pytest pytest-timeout`
2. ✅ 启动Redis: `redis-server`
3. ✅ 执行迁移: `python manage.py migrate assistant`
4. ✅ 创建目录: `mkdir -p media/ui_scripts media/workspaces`
5. ✅ 启动服务: `python manage.py runserver`
