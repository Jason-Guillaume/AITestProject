# 用户登录与会话令牌

## 文档目的

说明 **`POST /api/user/login/`** 的校验逻辑、返回的 **DRF Token** 形态，以及前端存储方式。说明 **无服务端登出接口** 时的令牌生命周期与风险面。

---

## 1. HTTP 路由与视图

- **路径**：`POST /api/user/login/`
- **视图**：`UserLoginAPIView`（`user/views.py`）
- **鉴权**：`authentication_classes = []`、`permission_classes = []`

---

## 2. 请求体与序列化器

**序列化器**：`UserLoginSerializer`（`user/serialize.py`）

| 字段 | 说明 |
|------|------|
| `username` | 必填；`strip` 后校验非空 |
| `password` | 必填 |
| `captcha_code` / `captcha_uuid` | **可选**；若任一非空则 **两者都必须有**，并按与注册相同的规则校验图形码且 **delete** 缓存 |

### 2.1 认证与账号状态

- `authenticate(username=username, password=password)`。
- 成功时检查：若用户 **`is_deleted`**（自定义字段）→ 拒绝「账号不可用」；若 **`not user.is_active`** →「该账号已被禁用」。
- 失败统一：**「账号或密码错误，请重试」**（不区分用户名不存在与密码错误，防枚举）。

### 2.2 Token 签发

```python
token, created = Token.objects.get_or_create(user=user)
```

- **模型**：`rest_framework.authtoken.models.Token`，表 **`authtoken_token`**。
- **语义**：同一用户重复登录返回 **已有** `token.key`（非每次轮换）；若需「单点踢人」需自行扩展（删除旧 Token 或改用 JWT）。

视图内对 `user.refresh_from_db()` 做 try/except，避免并发边缘问题。

## 3. 成功响应体

```json
{
  "code": 200,
  "msg": "登录成功",
  "data": {
    "token": "<DRF Token key>",
    "username": "...",
    "real_name": "...",
    "user_id": 1,
    "is_system_admin": false
  }
}
```

`is_system_admin` 来自自定义字段 **`User.is_system_admin`**，布尔化后返回。

## 4. 全局 API 认证（与登录配合）

项目使用 DRF **`TokenAuthentication`**（在 `settings.REST_FRAMEWORK` 中配置，以仓库为准）。前端业务请求通常在 Header 携带：

`Authorization: Token <token.key>`

具体拼接见 **`frontend/src/utils/request.js`** 拦截器。

---

## 5. 登出与令牌失效（重要）

**当前后端 `user` 应用未提供「注销 / 吊销 Token」的专用接口。**

前端 **`MainLayout.vue`** 中「退出登录」逻辑为：

- 确认后 **`localStorage.removeItem('token')`**（及 `username`、`user_id`、`real_name`、`is_system_admin` 等）
- **`router.push('/login')`**

即：**客户端丢弃令牌**；服务端 **`authtoken_token` 行仍存在**，在 Token 被删除前仍可被复用（若泄露则风险仍在）。高安全场景可增加：`POST /api/user/logout/` 删除 `request.user.auth_token` 或轮换密钥。

## 6. WebSocket 与 Token

压测、服务器日志等 WebSocket 使用查询参数 **`?token=<key>`**（见工程根 **`02_异步网关…`**），与登录返回的 **同一字符串** 一致。

## 7. 前端对接

- **`frontend/src/api/auth.js`**：`loginApi` 使用 **独立 axios 实例**（注释说明：避免与全局 `request` 拦截器循环依赖）。
- **`frontend/src/views/Login.vue`**：成功后将 `token`、`user_id`、`username`、`real_name`、`is_system_admin` 写入 `localStorage`。
