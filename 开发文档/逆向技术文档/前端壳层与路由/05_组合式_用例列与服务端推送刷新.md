# 组合式：用例类型列与服务端推送刷新

## 1. `useTestCaseTypeColumns`

- **文件**：`frontend/src/composables/useTestCaseTypeColumns.ts`
- **职责**：
  - **`getTypeExtraColumns(caseType)`**：按 **`functional | api | performance | security | ui-automation`** 返回 **表格附加短列**（方法、期望码、并发、RPS、框架等）。
  - **`getDrawerExtraFields(row)`**：抽屉内 **功能/安全** 长文本字段。
  - **`drawerApiHeadersText`** 等：API 头格式化。

## 2. 设计原则

- **主表只放短字段**；**URL、包名、定位符** 进抽屉，避免表格爆炸宽。

## 3. 与路由

- **`/test-case/:type`**：`type` 即列配置 key；**`TestCase.vue`** import 本组合式。

## 4. SSE / 轮询

- **本文件无网络**纯函数；列表刷新由 **`TestCase.vue`** 决定（可能 `getCasesApi` + 事件）。

## 5. 服务端推送刷新（概念）

- 若用例列表需 **多人协作** 实时更新，可：
  - 监听 **`useSseRefreshTrigger`** 的 **`buildUrl`** 指向 **用例变更流**（若后端提供）；
  - 或监听自定义 **`app:test-case-refresh`**（需在保存/导入成功后 `dispatchEvent`）。
- 当前仓库 **以页面实现为准**；本条目描述扩展点。

## 6. `streamGenerateAiCases` 与列表

- **`assistant.js`**：**fetch SSE** `/api/ai/generate-cases-stream/`；生成完成后 **`TestCase`** 应 `getCasesApi` 或增量插入。

## 7. HTTP 缓存

- 列表接口 **不加** `Cache-Control` 依赖浏览器；刷新靠 **手动/定时/SSE**。

## 8. WebSocket

- 用例列表 **默认无 WS**；**Execution** 的实时性在 **报告执行**、**流水线** 等模块。

## 9. 类型安全

- **`TestCaseRow`**：`@/types/testcase`；列 `cell: (r) => string` 强制字符串化。

## 10. 国际化

- 列 `label` 为中文硬编码；若 i18n，应改为 **`t('...')`**。

## 11. 性能用例列

- **`concurrency`、`duration_seconds`、`target_rps`** 右对齐数字，固定 `width` 减少抖动。

## 12. API 用例列

- **`api_method`、`api_expected_status`**：居中；与 **抽屉内完整 URL** 分离。

## 13. 小结

**列组合式 = 纯展示策略**；「推送刷新」归属页面数据层与 **`dashboard` 流式 API** 不同路。

## 14. 与 `useApiExecuteConsole` 边界

- 执行结果 **不改变** `getTypeExtraColumns`；可能更新 **`exec_count`** 等业务列若后端返回。

## 15. 测试

- 对每种 `caseType` 断言 **列数、key 唯一**。

## 附：联调自检（通用）

1. 浏览器 **Network** 中请求 URL 是否带 **`/api`** 前缀（`auth.js` 全路径除外）。
2. **`Authorization: Token …`** 是否随 `request` 发出；401 后是否退回登录页。
3. **Vite / Nginx** 是否同时代理 **`/api`** 与 **`/ws`**（若页面使用 WebSocket）。
4. 生产 **HTTPS** 下压测与日志是否使用 **`wss`**。

## 附：术语

- **REST**：与 **`request` axios 实例** 相关的 JSON HTTP 接口（`baseURL /api`）。
- **WS**：**`WebSocket`**，用于 k6 指标、SSH 日志 tail 等。
- **SSE**：**`EventSource`**，看板可用于触发摘要刷新。

- 静态资源与 **History 路由** 需 **fallback** 到 `index.html`。
