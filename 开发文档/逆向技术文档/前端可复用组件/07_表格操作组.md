# 可复用组件：表格操作组（TableActionGroup）

## 1. 文件

- **`frontend/src/components/common/TableActionGroup.vue`**

## 2. 用途

在 **`el-table`** 操作列中，把多个 **圆形图标按钮** 收敛为一行；超出 **`maxVisible`** 的放入 **下拉更多**。

## 3. Props

| Prop | 类型 | 默认 | 含义 |
|------|------|------|------|
| `row` | Object | `{}` | 当前行数据 |
| `actions` | Array | `[]` | 操作项列表 |
| `maxVisible` | Number | `3` | 直接展示数量 |

## 4. Action 项结构（约定）

每项建议包含：

- **`key`**：事件枚举值
- **`tooltip`**：悬浮说明 / 下拉文案
- **`icon`**：Element Plus 图标组件
- **`type`**：`primary` | `success` | `warning` | `danger` → 映射按钮样式类
- **`loading`** / **`disabled`**：可选

## 5. 事件

- **`@action`**：`(key, row) => void`**，父组件 **`switch(key)`** 调 API。

## 6. 使用页面示例

- **`EnvironmentManagement.vue`**：`envTableActions(row)` + **`handleEnvTableAction`**。
- **`performance/TaskManagement.vue`**：任务行 **`run`/`edit`/`delete`**。
- 其它管理员列表可复用，保持 **暗色主题** 样式一致。

## 7. 依赖

- **`MoreFilled`** 图标；**`el-tooltip`**、**`el-dropdown`**。

## 8. API

- 组件 **自身无 API**；仅 UI。

## 9. WebSocket

- **无**。

## 10. 可访问性

- 每个圆形按钮有 **tooltip**；键盘焦点顺序符合 DOM 顺序。

## 11. 样式要点

- `:deep(.table-action-group__btn)` 定义 **cyber** 边框与 hover 微动效。

## 12. 小结

**纯展示 + 事件委派**，降低表格操作列模板重复。

## 13. 扩展

- 若需 **权限指令** `v-permission`，可在父组件生成 **`actions`** 时过滤，而非改子组件。

## 14. 与 `el-link` 对比

- 定时任务列表页部分用 **`el-button link`** 而非本组件，属设计差异；新页择优统一。

## 15. 单元测试

- 挂载 N+1 个 action，断言 **可见数 = maxVisible**、**dropdown 项数 = N - maxVisible**。

## 附：联调自检（通用）

1. 浏览器 **Network** 中请求 URL 是否带 **`/api`** 前缀（`auth.js` 全路径除外）。
2. **`Authorization: Token …`** 是否随 `request` 发出；401 后是否退回登录页。
3. **Vite / Nginx** 是否同时代理 **`/api`** 与 **`/ws`**（若页面使用 WebSocket）。
4. 生产 **HTTPS** 下压测与日志是否使用 **`wss`**。

## 附：术语

- **REST**：与 **`request` axios 实例** 相关的 JSON HTTP 接口（`baseURL /api`）。
- **WS**：**`WebSocket`**，用于 k6 指标、SSH 日志 tail 等。
- **SSE**：**`EventSource`**，看板可用于触发摘要刷新。
