# 可复用组件：用例抽屉 API 面板（CaseDrawerApiPanel）

## 1. 文件

- **`frontend/src/components/testcase/CaseDrawerApiPanel.vue`**

## 2. 职责

在 **测试用例列表** 抽屉中展示 **API 类用例** 的额外字段：URL、Method、Header、Body 等，并内嵌 **执行/预览/AI 填数** 等动作，逻辑委托给 **`useApiExecuteConsole`**。

## 3. 组合式：`useApiExecuteConsole`

- **路径**：`frontend/src/composables/useApiExecuteConsole.ts`（体积大，含 cURL 解析、Prism 高亮、运行日志）。
- **入参**：`Ref<TestCaseRow | null>` 或兼容形态。
- **主要 API**（来自 `@/api/testcase`）：
  - **`runApiCaseApi`**：`POST /testcase/cases/{id}/run-api/`
  - **`previewRunApiCaseApi`**：`POST .../preview-run-api/`
  - **`aiFillTestDataApi`**：`POST /testcase/ai-fill-test-data/`
- **计划与报告**（`@/api/execution`）：**`getPlansApi`、`createReportApi、updateReportApi`** 用于部分「关联计划/报告」场景（以文件内引用为准）。

## 4. 环境与 URL

- 组合式内 **`useCurrentEnvironment()`**，拼装 **相对路径** 用例与 **`baseUrl`** 的最终请求 URL。

## 5. UI 结构提示

- 与 **`ApiExecuteConsoleDialog`** 共享同一 composable，避免重复实现执行管线。

## 6. WebSocket

- **无**。执行结果为 **HTTP 响应 + ExecutionLog 渲染**。

## 7. 类型

- **`TestCaseRow`**、**`ApiVariableExtractRule`**：`@/types/testcase`。

## 8. 失败与消息

- HTTP 错误靠 **`request`** 拦截器 + 组合式内 **`ElMessage`**（若调用）。

## 9. 与路由

- 用例列表路由 **`/test-case/:type`**（`TestCase.vue`）；抽屉不切换路由。

## 10. 性能

- Prism 语言包按需 import **json/php/...**；大 JSON 响应注意 **折叠或虚拟化**（若有）。

## 11. 小结

**CaseDrawerApiPanel = UI 壳 + useApiExecuteConsole**；逆向时优先读 composable。

## 12. SSE 说明

- 若抽屉内触发 **AI 流式**，应由 **`assistant.js` 的 `streamGenerateAiCases` 的 fetch** 处理；本面板以 **执行 API 用例** 为主，通常非 SSE。

## 13. 单元测试切入点

- Mock **`runApiCaseApi`** 返回 `execution_logs` 结构，断言 Prism 渲染开关。

## 14. 安全

- 预览/执行可能带 **真实 Token** 访问用户环境；仅登录可用。

## 15. 相关组件

- **`ApiExecuteConsoleDialog.vue`**（弹窗形态，同 composable）。

## 附：联调自检（通用）

1. 浏览器 **Network** 中请求 URL 是否带 **`/api`** 前缀（`auth.js` 全路径除外）。
2. **`Authorization: Token …`** 是否随 `request` 发出；401 后是否退回登录页。
3. **Vite / Nginx** 是否同时代理 **`/api`** 与 **`/ws`**（若页面使用 WebSocket）。
4. 生产 **HTTPS** 下压测与日志是否使用 **`wss`**。

## 附：术语

- **REST**：与 **`request` axios 实例** 相关的 JSON HTTP 接口（`baseURL /api`）。
- **WS**：**`WebSocket`**，用于 k6 指标、SSH 日志 tail 等。
- **SSE**：**`EventSource`**，看板可用于触发摘要刷新。
