# 03 — 大模型非流式连接测试（逆向）

## 1. `LlmTestConnectionAPIView`

- 路由：`POST /api/assistant/llm/test-connection/`（`assistant/urls.py`）。
- 权限：`IsAuthenticated`。
- Body：`api_key`（必填）、`model`（必填）、`base_url`（可选）。
- 目标解析：`_resolve_openai_target` + `OpenAI` SDK `_normalize_openai_sdk_base_url`（剥离 `/chat/completions` 后缀，兼容讯飞/通用 OpenAI 兼容网关）。
- 调用：`client.chat.completions.create`，system 使用模块级 `SYSTEM_PROMPT`，用户短句探活，`max_tokens=80`。
- **HTTP 状态码恒为 200**：业务码放在 `code` 字段（400/200），与部分前端拦截器约定一致。
- 错误：`OPENAI_SDK_AVAILABLE` 假时返回安装提示 `_openai_missing_response_json`；其它异常走 `resolve_ai_test_connection_openai_error`。

## 2. 与 `/api/ai/test-connection/` 的差异

- `AiTestConnectionAPIView`（`assistant/ai_urls`）走 `_guard_ai_request_or_429`，写 `AiUsageEvent`，更偏「产品内全局治理后的正式探测」。
- `LlmTestConnection` 更轻量，适用于助手侧自定义 Key 的快速校验（仍以登录为前提）。

## 3. 配置提示

- 智谱 GLM 未填 base_url 时可回落默认 host；非 GLM 模型若空 base_url 会提示必须填写自定义地址。

## 4. 安全

- Key 由客户端 POST，服务端不落库（除非另有系统设置模块）；传输应 HTTPS。
