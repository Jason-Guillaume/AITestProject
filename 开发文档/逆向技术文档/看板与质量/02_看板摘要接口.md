# 看板摘要接口

## 文档目的与读者

拆解 **`GET /api/execution/dashboard/summary/`**（**`DashboardSummaryAPIView`**）：**`project_id`** 下的用例/缺陷范围、**统计卡片**口径、**周/月折线**与 **活动流** 数据来源。适合对齐大屏数字与 ORM 实际含义、发现「饼图未按项目过滤」等边界。

## 1. 路由

**`GET /api/execution/dashboard/summary/`**（`execution/urls.py` → `DashboardSummaryAPIView`）。

---

## 2. Query 与管理员判定

| 参数 | 说明 |
|------|------|
| **`project_id`** | 可选 **字符串**；若能 **`int()`** 则用于过滤；非法时 **`_case_scope_qs`** 等对用例侧返回 **`qs.none()`** 等（以分支为准） |

**`is_admin`**：`superuser` **或** `staff` **或** **`user.is_system_admin`**。

---

## 3. 数据范围：`_case_scope_qs`

- 基：**`TestCase.objects.filter(is_deleted=False)`**。
- **传了合法 `project_id`**：**`module__project_id=pid`**。
- **未传 `project_id`** 且用户 **已登录且非管理员**：**`Q(module__project__members=user) | Q(creator=user)`** `.distinct()`。
- **未传且管理员或未登录**：**不过滤项目**（全表未删用例）。

---

## 4. 数据范围：`_defect_scope_qs`

- 基：**`TestDefect.objects.filter(is_deleted=False)`**。
- **合法 `project_id`**：依次尝试 **`project_id`**、**`testcase__module__project_id`**、**`test_case__module__project_id`**（**`hasattr` 动态判断**）。
- **未传 `project_id` 且普通用户**：**`creator=user`** 或 **经 testcase 链路的 `module__project__members`**（`.distinct()`）。
- **管理员或未登录未传参**：不过滤（全表）。

---

## 5. 统计卡片（`statCards`）

| 键 | 口径摘要 |
|----|-----------|
| **`totalCases`** | **`_case_scope_qs().count()`** |
| **`todayExecuted`** | **`TestReport`** 今日 **`create_time`** 窗口内条数（**近似「今日执行」**，非 ExecutionLog 逐条数）；带 `project_id` 时按 **`project_id`** 或 **`plan__version__project_id`** 过滤 |
| **`unresolvedDefects`** | **`_defect_scope_qs().exclude(status=4)`**（**4=已关闭**） |
| **`passRate`** | 存在 **`plan_status=3`** 的 **`TestPlan`** 时，对其 **`pass_rate`** 做 **`Avg`**；否则 **0**；展示为 **整数百分比字符串** |

**Delta 字段**（与昨日/今日对比）：

- 用例：**今日新建 vs 昨日新建** 用例数差。
- 报告：今日 vs 昨日 **报告创建数** 差。
- 缺陷：今日新增未解决 vs 昨日新增未解决 差。

**`barWidth`**：由 **delta** 推导的 **百分比字符串**（用于前端装饰条，非真实进度条语义）。

## 6. 图表与活动流

| 区域 | 逻辑 |
|------|------|
| **`lineChart.week/month`** | **`TestReport` / `TestDefect`** 在 **7/30 天** 内按 **`TruncDate(create_time)`** 分组 **`Count`** |
| **`pieChart`** | 按 **`TestDefect.severity`** 1–4 映射到 **安全/合规/性能/功能** 标签；**注意**：当前实现为 **`TestDefect.objects.filter(is_deleted=False, severity=sev)`**，**未套用 `_defect_scope_qs`**，与卡片/折线 **权限口径可能不一致** |
| **`barChart`** | 近 **7 天** 缺陷 **新增** 数（与 week 折线缺陷序列一致） |
| **`activities`** | 最多 **6** 条：优先 **进行中计划**（2）、**新缺陷 status=1**（2）、**运行中 PerfTask**（1），不足则补 **已完成计划**；**未按 `project_id` 过滤**（全库时间排序切片） |

## 7. 其它工具方法

- **`_humanize_delta(dt)`**：相对时间中文文案（秒/分/小时/天）。
- **`_last_n_days(n)`**：本地日期列表。
- **`_count_by_day(model_cls, field_name, dates)`**：**`is_deleted=False`** + 日期范围 + **`TruncDate`** 聚合。
