# 质量看板接口

## 文档目的与读者

说明 **`GET /api/execution/dashboard/quality/`**（**`QualityDashboardView`**）的 **日期区间**、**缓存键**、**`MetricCalculator` 逐日 upsert** 与 **响应结构**；并标注 **`filters.time_range` 写死** 等实现细节，便于与前端联调及后续修 bug 时对照。

---

## 1. 路由与参数

| Query | 说明 |
|--------|------|
| **`project_id`** | 可选；**非空且非整数** → **400** `{"msg": "project_id 必须为整数"}` |
| **`time_range`** | 可选，默认 **`30d`**；合法值 **`7d`/`14d`/`30d`/`90d`**（大小写不敏感，经 **`strip().lower()`**）；其它值 **回落 30 天**（`_resolve_dates` 内 `days_map.get(tr, 30)`） |

**`_resolve_dates`**：返回 **`start_date`/`end_date`**（**`localdate`** 闭区间）及 aware **`start_dt`/`end_dt`**（当前 **`get` 主流程未使用**后两个返回值，仅解构占位）。

---

## 2. 缓存

- **Key**：`quality_dashboard_metric:{pid或null}:{start}:{end}:{time_range}`  
  其中 **`pid`** 为 **`int`** 或 **`None`** 时字符串 **`null`**。
- **TTL**：**60 秒**（`cache.set(..., timeout=60)`）。
- **命中**：直接 **`return Response(cached)`**，不再跑计算器。

---

## 3. 指标计算（逐日同步）

对 **`[start_date, end_date]`** 内 **每一天** `cur`：

1. **`MetricCalculator.calc_pass_rate(pid, cur)`**
2. **`calc_defect_density(pid, cur)`**
3. **`calc_requirement_coverage(pid, cur)`**

详见《04》。首次请求 **长区间**（如 90 天）会 **连续 upsert 90 次**，可能较慢，依赖缓存缓解。

---

## 4. 读取 `TestQualityMetric`

```python
metrics_qs = TestQualityMetric.objects.filter(
    is_deleted=False,
    metric_date__gte=start_date,
    metric_date__lte=end_date,
).filter(dimension__project_id=(pid if pid is not None else None))
```

- **`pid` 有值**：按 **`dimension.project_id`** 与项目对齐。
- **`pid` 为 `None`**：过滤 **`dimension__project_id=None`**。  
  而 **`MetricCalculator._dimension(None)`** 写入的是 **`{"project_id": "all"}`**（字符串 **`"all"`**），与 **`None`** **可能不一致**，若全平台视图 **折线全 0**，需核对 ORM 对 JSONField 的匹配行为或视为已知实现风险。

## 5. 响应结构摘要

| 区块 | 内容 |
|------|------|
| **`filters`** | **`project_id`**、**`start_date`/`end_date`**（ISO 日期字符串） |
| **`filters.time_range`** | **源码写死为 `"30d"` 字符串**（与请求 **`time_range`** 可能不一致，属实现细节；前端应以请求参数或 `start/end` 为准） |
| **`trendChart`** | **`xAxis`** + 三条 **`series`**（通过率%、缺陷密度、需求覆盖率%） |
| **`latestMetrics`** | 取 **最后一天** 的三个指标值 |
| **`raw`** | 分类型 **`points`** 列表，含 **`metric_type` 常量** |
| **`chartsCompat`** | 拆成三条单序列趋势，兼容旧前端字段名 |
