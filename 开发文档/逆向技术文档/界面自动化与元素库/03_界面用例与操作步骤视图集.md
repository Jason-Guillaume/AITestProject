# 03 — 界面用例与操作步骤视图集（逆向）

## 1. `UITestCaseViewSet`

- 路由：`/api/ai/ui-test-cases/`（ModelViewSet CRUD + 列表筛选）。
- 认证：`TokenAuthentication` + `IsAuthenticated`。
- `get_queryset`：可选 `module_id`（**注意**：此处 `module` 指 `UIModule` FK，不是 `testcase.TestModule`）。
- 排序：`order, id`。

## 2. `UIActionStepViewSet`

- 路由：`/api/ai/ui-action-steps/`。
- 过滤：`test_case_id`（关联 `UITestCase`）。
- 排序：`sequence`（步骤序）。

## 3. 与业务测试用例的关系

- `assistant.ui_element_models.UITestCase` 为 **元素库产品内** 的 UI 场景编排，不等同于 `testcase.models.TestCase`（后者为全平台正式用例与执行引擎对象）。
- 若需要将 UI 元素库场景导出为正式用例，应通过定制转换逻辑（代码库中若有 bridges 需另文说明）。

## 4. 使用场景

## 5. API 风格

- 错误响应部分为英文（`error` 字段），与旧版前端一致；新前端应统一收口提示文案。
