## 1. 路由与类

- `POST /api/ai/ui-automation/generate/`（`assistant/ai_urls.py`）。
- 视图：`assistant/ui_automation_views.py` → `UiAutomationGenerateAPIView`。
- 鉴权：`TokenAuthentication` + `IsAuthenticated`；`JSONParser`。

## 2. 请求体

- `url`：必填，strip 后不可为空。
- `steps`：必填，步骤描述字符串 strip 后不可为空。

## 3. 行为（当前实现）

- **不做真实 LLM 调用**：按模板字符串拼接一段 Python `unittest` + `selenium` 示例代码。
- 代码结构：`setUpClass` 启动 Chrome、`test_user_scenario` 等待 body、查找基础控件、`test_page_load_performance` 简单计时。
- 响应：`{ success: True, code: <多行字符串 Python>, message: "...（Mock）" }`。

## 4. 设计意图（注释）

- 标注为 Mock：用于前端联调 / 演示「从自然语言步骤 → 脚本骨架」的占位实现；后续可替换为真正的大模型或规则引擎生成。

## 5. 风险点

- `url.split('://')[1]...` 在畸形 URL 时可能抛错——当前假定合法 `http(s)://`。
- 未对 `steps` 做 XSS/注入清洗，因结果为代码字符串而非直接执行。

## 6. 演进方向

- 接入元素库（`UIPageElement`）按 locator 生成更可靠脚本；与 `UIScriptUpload` 执行链路打通。
