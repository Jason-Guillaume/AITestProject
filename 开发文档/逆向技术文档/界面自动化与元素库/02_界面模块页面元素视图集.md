# 02 — 界面模块、页面、元素视图集（逆向）

## 1. 路由前缀

- 挂载在 `assistant/ai_urls.py` 的 `DefaultRouter`：
  - `/api/ai/ui-modules/`
  - `/api/ai/ui-pages/`
  - `/api/ai/ui-elements/`
  - `/api/ai/ui-test-cases/`
  - `/api/ai/ui-action-steps/`

## 2. `UIModuleViewSet`

- 认证：`TokenAuthentication` + `IsAuthenticated`。
- `get_queryset`：可按 `project_id`、`parent_id` 过滤；`parent_id` 为 `null`/`0` 时取根模块。
- **自定义** `GET .../tree/?project_id=`：强制要求 `project_id`，序列化根模块为 `UIModuleTreeSerializer` 树。

## 3. `UIPageViewSet`

- 过滤：`module_id`；排序 `order,id`。

## 4. `UIPageElementViewSet`

- 过滤：`page_id`、`module_id`（经 `page__module_id`）、`search`（`name/description/locator_value` icontains OR）。
- **自定义** `GET .../by_name/?name=&page_id?`：`name` 必填；`name__iexact` 匹配；可选限定 `page_id`。用于 AI 生成或录制回放时按「元素昵称」解析 locator。
- 404：未找到返回英文 error（历史接口风格）。

## 5. 模型分层（POM）

- `UIModule` → `UIPage` → `UIPageElement`：与经典 Page Object 一致，`order` 字段支持拖拽排序展示。

## 6. 权限现状

- 仅认证，不含项目成员细粒度隔离——若需多租户应扩展 `get_queryset` 校验 `project.members`。
