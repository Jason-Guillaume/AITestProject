# 系统审计事件列表与逗号分隔导出

## 文档目的

说明 **`GET /api/sys/audit/events/`** 分页列表与 **`GET /api/sys/audit/export.csv`** 流式导出；与 **个人审计** **`/api/user/me/audit/events/`** 的差异（全库 vs `creator=self`）；导出同样调用 **`record_export_audit`**。

---

## 1. 列表：`AuditEventListAPIView`

**路径**：`GET /api/sys/audit/events/`  
**权限**：`[IsAuthenticated, IsSystemAdmin]`

### 1.1 查询参数

| 参数 | 说明 |
|------|------|
| `action` | `create` / `update` / `delete` / `export` / `execute`（与 `AuditEvent.ACTION_*` 一致） |
| `object_app` / `object_model` / `object_id` | 精确过滤 |
| `start_date` / `end_date` | `YYYY-MM-DD`，作用于 **`create_time` 日期** |
| `page` | 默认 1，最小 1 |
| `page_size` | 默认 50，**1～200** |

### 1.2 查询集

`AuditEvent.objects.filter(is_deleted=False)` — **不限定 creator**，即 **系统级全量**（含各用户产生的审计行）。

### 1.3 返回字段

`values` 投影含：`id`、`action`、`object_*`、`request_path`、`ip`、`user_agent`、**`before`、`after`、`extra`**、`creator_id`、`create_time`（比个人审计接口 **更全**，含快照 JSON）。

外层：`{"code":200,"data":{"page","page_size","total","items"}}`。

---

## 2. CSV 导出：`AuditEventExportCsvAPIView`

**路径**：`GET /api/sys/audit/export.csv`

| 参数 | 默认 | 范围 |
|------|------|------|
| `limit` | 5000 | 1～200000 |

- 筛选与列表相同（**无** `action` 逗号多值文档化需求，代码中为 **单值** `action` 精确匹配）。
- 列：`create_time, action, object_app, object_model, object_id, object_repr, request_path, ip, creator_id` — **不含** `before`/`after` 大字段（控制 CSV 体积）。
- **`StreamingHttpResponse`** + `iterator(chunk_size=1000)`。
- **`record_export_audit(..., extra={"export":"audit_events","rows": limit})`**。

---

## 3. 与个人审计对比

| 维度 | `/api/sys/audit/events/` | `/api/user/me/audit/events/` |
|------|---------------------------|-------------------------------|
| 权限 | 系统管理员 | 任意登录用户 |
| 范围 | 全库未删 | **`creator=request.user`** |
| `before`/`after` | 列表中带出 | 个人接口 **不返回** |
