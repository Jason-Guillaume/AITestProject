## 2. 列表与创建：`AiQuotaPolicyListCreateAPIView`

**路径**：`GET|POST /api/sys/ai-quota/policies/`  
**权限**：`[IsAuthenticated, IsSystemAdmin]`

### 2.1 GET 查询参数

| 参数 | 作用 |
|------|------|
| `scope_type` | 精确过滤 |
| `project_id` / `org_id` / `user_id` | 整数过滤；非法字符串忽略 |
| `limit` | 返回条数上限，默认 **200**，范围 **1～500** |

结果：`{"code":200,"msg":"ok","data":[...]}`，元素为 **`AiQuotaPolicySerializer`** 序列化结果（含 `creator_name` 等）。

### 2.2 POST

- **`AiQuotaPolicySerializer(data=..., context={"request": request})`**
- **`ser.save(creator=request.user, updater=request.user)`**

## 3. 详情：`AiQuotaPolicyDetailAPIView`

**路径**：`GET|PUT|DELETE /api/sys/ai-quota/policies/<policy_id>/`  
**权限**：同上。

| 方法 | 行为 |
|------|------|
| GET | `pk` + `is_deleted=False`；无则 **404** |
| PUT | `partial=True`；`before`/`after` 全量序列化快照写审计 **`ACTION_UPDATE`** |
| DELETE | **`is_deleted=True`** + `updater`；审计 **`ACTION_DELETE`**（`before` 快照） |
