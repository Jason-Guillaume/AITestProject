# 测试方案与设计数据模型

## 文档目的

集中列出 **`TestApproach`**、**`TestApproachImage`**、**`TestDesign`**、**`TestModule`** 四张（组）表及与 **`TestProject` / 用例类型枚举** 的关系，供迁移与序列化字段对照。

---

## 1. `TestApproach`（`test_approach`）

| 字段 | 类型/说明 |
|------|-----------|
| `scheme_name` | 方案名称 |
| `version` | 版本号字符串 |
| `cover_image` | **CharField**：封面展示用 URL/路径，非 ImageField |
| `test_goal` | 目标长文本 |
| `test_category` | 1～4 单元/集成/系统/用户验收 |
| BaseModel | `creator`、`updater`、时间、`is_deleted` |

---

## 2. `TestApproachImage`（`test_approach_image`）

| 字段 | 说明 |
|------|------|
| `approach` | FK → `TestApproach`，`related_name="images"` |
| `image` | `ImageField`，`approach_images/` |
| `sort_order` | 排序权重 |

**排序**：默认 **`sort_order` 升、`create_time` 降**。

---

## 3. `TestDesign`（`test_design`）

| 字段 | 说明 |
|------|------|
| `design_name` | 名称 |
| `req_count` / `point_count` / `case_count` | 统计型整数，默认 0 |
| `review_status` | 1 未评审 / 2 评审中 / 3 已评审 |
| `archive_status` | 1 未归档 / 2 已归档 |
| BaseModel | 同上 |

与 **`TestCase`** 的关联不在此表直接体现，多通过业务外键或中间表（见用例分册）。

---

## 4. `TestModule`（`test_module`）

| 字段 | 说明 |
|------|------|
| `project` | FK → `TestProject` |
| `name` | 模块名 |
| `parent` | 自关联，树 |
| `test_type` | 与 **`TEST_CASE_TYPE_*`** 枚举一致 |
