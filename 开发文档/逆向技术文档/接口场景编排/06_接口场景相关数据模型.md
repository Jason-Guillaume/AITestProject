# 接口场景相关数据模型

## 文档目的与读者

汇总 **`api_scenario`**、**`api_scenario_step`**、**`api_scenario_run`**、**`api_scenario_step_run`** 四表关系、级联、索引与关键枚举，便于与 ViewSet/Runner 文档交叉查阅。

---

## 1. ER 关系（逻辑）

```
TestProject (1) ──< (N) ApiScenario
TestEnvironment (1) ──< (N) ApiScenario  [可选 FK，SET_NULL]

ApiScenario (1) ──< (N) ApiScenarioStep ──> (1) TestCase [PROTECT]
ApiScenario (1) ──< (N) ApiScenarioRun

ApiScenarioRun (1) ──< (N) ApiScenarioStepRun ──> (1) ApiScenarioStep [PROTECT]
```

**`ApiScenarioRun.environment_id`**、**`ApiScenarioStepRun.test_case_id` / `execution_log_id`**：以 **整型快照** 引用外部表，**无 Django FK**（跨 app 或历史原因）。

---

## 2. `api_scenario`

| 字段 | 说明 |
|------|------|
| **`project_id`** | CASCADE 随项目删 |
| **`name`** | varchar(255) |
| **`environment_id`** | 可空，**`SET_NULL`** → **`testcase.TestEnvironment`** |
| **`default_variables`** | JSON，场景级变量池 |
| **`failure_strategy`** | **`abort`** / **`continue`** |
| **`is_active`** | 为假时 **`run` 拒绝** |
| BaseModel | 软删、creator 等 |

**索引**：**`(project, -create_time)`**。**ordering**：**`-create_time`**。

---

## 3. `api_scenario_step`

| 字段 | 说明 |
|------|------|
| **`scenario_id`** | CASCADE |
| **`order`** | 执行顺序 |
| **`name`** | 步骤名 |
| **`test_case_id`** | **PROTECT** |
| **`is_enabled`** | 禁用则运行跳过 |
| **`failure_strategy`** | 空=继承场景；否则 abort/continue |
| **`extraction_rules`** | JSON 数组 |
| **`step_overrides`** | JSON 对象 |

**索引**：**`(scenario, order)`**、**`(test_case)`**。  
**ordering**：**`order`, `id`**。

---

## 4. `api_scenario_run`

| 字段 | 说明 |
|------|------|
| **`scenario_id`** | CASCADE |
| **`status`** | pending / running / success / failed / partial |
| **`trace_id`** | 索引字段，最长 36 |
| **`started_at` / `finished_at` / `duration_ms`** | 运行窗口 |
| **`environment_id`** | 正整数或空 |
| **`initial_variables` / `final_variables` / `summary`** | JSON |
| **`celery_task_id`** | varchar(64) |
| **`error_message`** | text |

**索引**：**`(scenario, -create_time)`**、**`(trace_id)`**。

---

## 5. `api_scenario_step_run`

| 字段 | 说明 |
|------|------|
| **`run_id`** | CASCADE |
| **`step_id`** | **PROTECT** |
| **`order` / `test_case_id`** | 快照 |
| **`execution_log_id`** | 可空 |
| **`status`** | completed / failed / skipped |
| **`passed`** | bool |
| **`extracted_variables`** | JSON |
| **`message`** | varchar(255) |

**索引**：**`(run, order)`**、**`(test_case_id)`**。  
**ordering**：**`order`, `id`**。

---

## 6. 数据范围补充（列表 API）

- **`ApiScenario`**：经 **`project`** 成员过滤。
- **`ApiScenarioRun` / `ApiScenarioStepRun`**：无 **`project`** 等关系字段时，**`BaseModelViewSet`** 兜底 **`creator=user`**（非管理员）；跨角色查看运行历史可能受限，见《03》。
