## 🚀 如何使用

### 访问路径
```
侧边栏 → 🤖 智能体中心 → 元素库管理
```

### 基本操作
1. 创建模块 → 创建页面 → 添加元素
2. 配置元素的定位方式和定位表达式
3. 在 AI 脚本生成器中自动使用元素库

---

## 📊 API 端点

```
GET    /api/ai/ui-modules/tree/         # 获取树形结构
GET    /api/ai/ui-elements/?page_id={id} # 获取元素列表
GET    /api/ai/ui-elements/by_name/      # 按名称查找元素
POST   /api/ai/ui-modules/              # 创建模块
POST   /api/ai/ui-pages/                # 创建页面
POST   /api/ai/ui-elements/             # 创建元素
```
