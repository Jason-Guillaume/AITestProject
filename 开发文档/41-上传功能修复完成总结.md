## 📡 API集成

**接口**: POST /api/assistant/ui-scripts/

**单文件上传参数**:
```javascript
{
  name: "脚本名称",
  script_type: "LINEAR",
  file: File对象
}
```

**ZIP导入参数**:
```javascript
{
  name: "脚本名称",
  script_type: "POM",
  entry_point: "tests/test_login.py",
  file: File对象
}
```

**Git导入参数**:
```javascript
{
  name: "脚本名称",
  script_type: "POM",
  entry_point: "tests/test_login.py",
  git_repo_url: "https://github.com/user/repo.git"
}
```

## 🎯 用户操作流程

### 单文件上传 (.py)
1. 访问工作台页面
2. 选择"单文件上传(线性)"
3. 输入脚本名称
4. 拖拽或点击上传 .py 文件
5. 点击"上传脚本"按钮
6. 等待上传完成
7. 查看成功提示

### ZIP包导入
1. 选择"ZIP/Git导入(POM)"
2. 输入脚本名称
3. 选择"ZIP 文件"
4. 上传 .zip 文件
5. 输入执行入口点（如：tests/test_login.py）
6. 点击"导入脚本"按钮
7. 查看成功提示

### Git仓库导入
1. 选择"ZIP/Git导入(POM)"
2. 输入脚本名称
3. 选择"Git 仓库"
4. 输入Git仓库URL
5. 输入执行入口点
6. 点击"导入脚本"按钮
7. 查看成功提示

## 🔍 技术细节

### FormData构建
```javascript
const formData = new FormData();
formData.append('name', linearConfig.value.name);
formData.append('script_type', 'LINEAR');
formData.append('file', linearConfig.value.file);
```

### Axios配置
```javascript
await axios.post('/api/assistant/ui-scripts/', formData, {
  headers: {
    'Content-Type': 'multipart/form-data'
  }
});
```

### Element Plus Upload组件
```vue
<el-upload
  :auto-upload="false"
  :on-change="handleLinearFileChange"
  :on-remove="handleLinearFileRemove"
  :file-list="linearConfig.file ? [{ name: linearConfig.file.name }] : []"
>
```
