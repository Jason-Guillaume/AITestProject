## API接口测试

### 测试1：上传单个Python文件（LINEAR模式）

**请求**:
```bash
curl -X POST http://localhost:8000/api/assistant/ui-scripts/ \
  -H "Authorization: Token YOUR_TOKEN" \
  -F "name=登录测试脚本" \
  -F "script_type=LINEAR" \
  -F "entry_point=test_login.py" \
  -F "file_path=@test_login.py"
```

**测试文件内容** (`test_login.py`):
```python
from selenium import webdriver
from selenium.webdriver.common.by import By

def test_login():
    driver = webdriver.Chrome()
    driver.get("https://example.com/login")
    driver.find_element(By.ID, "username").send_keys("admin")
    driver.find_element(By.ID, "password").send_keys("password")
    driver.find_element(By.ID, "submit").click()
    assert "Dashboard" in driver.title
    driver.quit()

if __name__ == "__main__":
    test_login()
```

### API端点

- `POST /api/assistant/ui-scripts/` - 创建脚本上传
- `GET /api/assistant/ui-scripts/` - 获取脚本列表
- `GET /api/assistant/ui-scripts/{id}/` - 获取脚本详情
- `PUT /api/assistant/ui-scripts/{id}/` - 更新脚本
- `DELETE /api/assistant/ui-scripts/{id}/` - 删除脚本
- `POST /api/assistant/ui-scripts/{id}/toggle_active/` - 切换启用状态
- `GET /api/assistant/ui-scripts/{id}/workspace_info/` - 获取工作空间信息
