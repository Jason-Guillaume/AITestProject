# 项目功能总览

本项目是一个 **AI 辅助 UI 自动化测试平台**，覆盖从 **知识库构建 → 语义检索 → AI 用例生成 → UI 脚本管理与执行 → 缺陷管理 → 项目/任务调度 → 前端展示** 的完整业务链路。下面提供对所有子系统的快速索引与简要说明，详细实现请参考同目录下的各模块文档。

---

## 1. 公共基础模块 (`common`)
- 提供 `BaseModel`、统一异常 `BusinessException`、统一响应 `ApiResponse`、通用序列化器 `BaseModelSerializers` 与通用视图 `BaseModelViewSet`。
  - 详见 `01_公共基础模块.md`

## 2. 知识库与向量检索 (`assistant` 知识库子模块)
- 文档上传、切块、向量化（Chroma）、语义相似度检索（RAG），为后续 AI 生成提供上下文。
  - 详见 `02_知识库与向量检索.md`

## 3. AI 生成与提示构建 (`assistant` AI 核心)
- 基于 Prompt Engineering 调用 LLM（OpenAI / Zhipu），生成 API/功能/安全用例；包括结果清洗、去重、持久化。
  - 详见 `03_AI生成与提示构建.md`

## 4. UI 自动化模块 (`assistant` UI 子模块)
- UI 元素树形模型（模块、页面、元素、用例、步骤），脚本上传、异步执行、WebSocket 实时日志推送。
  - 详见 `04_UI自动化模块.md`

## 5. 缺陷管理模块 (`defect`)
- 缺陷 CRUD、批量软/硬删除、回收站、从执行或安全扫描自动生成缺陷。
  - 详见 `05_缺陷管理模块.md`

## 6. 执行与调度模块 (`execution`)
- 测试计划、报告、性能任务、定时任务、API 场景化执行；使用 Celery 异步、APScheduler 定时、Channels 实时推送。
  - 详见 `06_执行与调度模块.md`

## 7. 前端概述 (`frontend`)
- 基于 Vue 3 或 React 18 的单页面应用，使用 Pinia/Redux、Ant Design、ECharts、WebSocket、JWT+HttpOnly Cookie。
  - 详见 `07_前端概述.md`

## 8. 项目管理模块 (`project`)
- 项目、任务、发布计划管理，支持批量操作、CSV 导出、进度计算。
  - 详见 `08_项目管理模块.md`

## 9. 用户与组织模块 (`user`)
- 用户注册/登录、组织成员管理、系统消息、验证码、AI 模型配置、细粒度权限控制。
  - 详见 `09_用户与组织模块.md`

## 10. 其他辅助目录
| 目录 | 说明 |
|------|------|
| `memory/` | 运行时缓存、任务锁等实现细节。 |
| `data/` | 数据导入/导出、种子文件。 |
| `reports/` | 自动生成的 PDF/HTML 报告、质量仪表盘快照。 |
| `scripts/` | 部署、迁移、初始化脚本；如 `start-all.bat`。 |
| `server_logs/` | 后端、Celery、nginx 等日志。 |
| `htmlcov/` | 单元测试覆盖率报告。 |
| `templates/` | Django 邮件、系统消息模板。 |
| `testcase/` | 手工编写的 UI/接口测试用例。 |
| `tests/` | 单元/集成测试，覆盖核心功能。 |

---

## 使用与扩展指引
1. **阅读文档**：先打开 `10_功能总览.md` 了解整体结构，再根据需求跳转到对应模块文档。
2. **排查错误**：每个模块文档末尾都有 **常见错误与排查** 小结，遇到问题可直接查阅。
3. **新增功能**：在对应子系统下新增模型、序列化器、视图后，创建对应的 Markdown 文档并在本总览中添加索引。
4. **脚本入口**：`manage.py` 用于 Django 管理，`start-all.bat` 一键启动后端、前端、Celery、Redis 等依赖。
