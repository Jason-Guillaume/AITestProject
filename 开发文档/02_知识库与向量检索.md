# 知识库与向量检索（assistant 知识库子模块）

## 功能概述
实现文档的上传、切块、向量化、持久化存储以及基于语义相似度的检索（RAG），为 AI 生成提供上下文支撑。

## 使用技术
- **Django ORM**：存储 `KnowledgeArticle`、`KnowledgeDocument` 的元数据。
- **ChromaDB**：本地持久化向量数据库，提供向量存储与相似度搜索。
- **Embedding 模型（Zhipu / OpenAI）**：将文本块转换为向量。
- **Celery**：后台异步执行文档切块、向量化等耗时任务。
- **Django Channels（可选）**：实时推送向量化进度。

## 关键代码文件及片段
### `assistant/models.py`
```python
class KnowledgeArticle(BaseModel):
    title = models.CharField(max_length=255)
    content = models.TextField()

class KnowledgeDocument(BaseModel):
    file = models.FileField(upload_to="knowledge_docs/")
    md5 = models.CharField(max_length=32)
```

### `assistant/knowledge_rag.py`
```python
def _chroma_path() -> str:
    return os.path.join(settings.BASE_DIR, "chroma_db")

def get_collection():
    client = chromadb.PersistentClient(path=_chroma_path())
    return client.get_or_create_collection(name="knowledge")
```

### 检索实现
```python
def search_similar(text: str, top_k: int = 5):
    collection = get_collection()
    embedding = embed_single(text)
    results = collection.query(query_embeddings=[embedding], n_results=top_k)
    return results
```

### 异步任务（Celery）
```python
@shared_task
def process_knowledge_document_task(doc_id: int):
    doc = KnowledgeDocument.objects.get(id=doc_id)
    raw = doc.file.read().decode()
    chunks = _chunk_text(raw)
    embeddings = embed_batch(chunks)
    collection = get_collection()
    ids = [f"{doc_id}_{i}" for i in range(len(chunks))]
    collection.add(ids=ids, embeddings=embeddings, documents=chunks)
```

## 为什么使用这些技术
- **Chroma** 轻量、易部署，适合内部私有化部署的企业。
- **Embedding** 提供语义相似度，比传统关键词搜索更精准，提升 AI 生成的上下文相关性。
- **Celery** 确保文档处理的耗时操作不阻塞 API，提升系统并发性。

## 常见错误与排查
1. **向量库路径错误**：`_chroma_path` 指向的目录不存在导致初始化失败，检查项目根目录是否存在 `chroma_db`，必要时手动创建。
2. **Embedding 服务不可达**：`embed_single` 抛网络异常，检查 `settings.py` 中的 `EMBEDDING_ENDPOINT`、`API_KEY` 是否正确。
3. **Celery 任务未执行**：未启动 `celery -A AITestProduct worker -l info`，确认消息中间件（Redis/RabbitMQ）已运行。
4. **检索返回空**：文档可能未成功写入向量库，查看 `process_knowledge_document_task` 的日志，确保 `collection.add` 成功且 `ids` 唯一。
