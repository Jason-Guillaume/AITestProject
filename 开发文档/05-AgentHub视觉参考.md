## 🎨 颜色方案

### 主题色
```css
/* 背景渐变 */
background: linear-gradient(180deg, #0a0e1a 0%, #1a1f2e 100%);

/* 卡片背景 */
background: rgba(255, 255, 255, 0.03);

/* 边框 */
border: 1px solid rgba(100, 255, 218, 0.12);

/* 主色调 - 青色 */
--primary: #00d8ff;

/* 文字颜色 */
--text-primary: #e2e8f0;
--text-muted: rgba(148, 163, 184, 0.9);
```

### 智能体图标渐变色

| 智能体 | 渐变色 | 预览 |
|--------|--------|------|
| 需求评审助手 | `#667eea → #764ba2` | 紫色系 |

| UI 自动化脚本 | `#4facfe → #00f2fe` | 蓝色系 |
| 接口文档分析 | `#43e97b → #38f9d7` | 绿色系 |

## 🎭 交互效果

### 卡片悬停效果
```
正常状态:
- transform: translateY(0)
- border: rgba(100, 255, 218, 0.12)
- box-shadow: none

悬停状态:
- transform: translateY(-4px)          ← 向上抬起 4px
- border: rgba(0, 216, 255, 0.4)       ← 边框变亮
- box-shadow: 0 8px 32px rgba(0, 216, 255, 0.2)  ← 霓虹光晕
- 内部光晕效果显示
```

### 按钮动画
```
正常状态:
- 箭头位置: translateX(0)

悬停状态:
- 箭头位置: translateX(4px)           ← 箭头向右移动
- 按钮背景: 渐变色加深
- 阴影增强
```
