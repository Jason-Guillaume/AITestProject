# AITestProduct 项目配置模块

## 模块概述
项目根配置目录，包含 Django 项目的核心配置文件。

## 文件列表

### __init__.py
- **作用**: Python 包初始化文件，使该目录成为 Python 包

### asgi.py
- **作用**: ASGI (Asynchronous Server Gateway Interface) 配置文件
- **业务功能**: 
  - 配置 Daphne WebSocket 服务器
  - 集成 Channels 路由
  - 支持实时消息推送（执行进度、日志流等）

### celery.py
- **作用**: Celery 分布式任务队列配置
- **业务功能**:
  - 初始化 Celery 应用
  - 配置 Redis/MySQL 作为消息代理
  - 配置任务序列化方式（JSON）
  - 支持异步执行测试任务、AI 生成等耗时操作

### settings.py
- **作用**: Django 核心配置文件
- **业务功能**:
  - 数据库配置 (MySQL)
  - Redis 缓存配置
  - Channels 层配置
  - REST Framework 配置
  - Celery 配置
  - AI 相关配置 (RAG、配额、并发限制)
  - 服务器日志配置 (SSH、Loki、Elasticsearch)
  - 邮件配置
  - 加密密钥配置

### urls.py
- **作用**: 主路由配置文件
- **业务功能**: 分发 API 请求到各子应用
  - `/admin/` - Django 后台
  - `/api/sys/` - 系统管理
  - `/api/user/` - 用户中心
  - `/api/project/` - 项目管理
  - `/api/testcase/` - 测试用例
  - `/api/execution/` - 测试执行
  - `/api/defect/` - 缺陷管理
  - `/api/assistant/` - AI 助手
  - `/api/server-logs/` - 服务器日志

### wsgi.py
- **作用**: WSGI (Web Server Gateway Interface) 配置文件
- **业务功能**: 传统同步 Web 服务器入口点
