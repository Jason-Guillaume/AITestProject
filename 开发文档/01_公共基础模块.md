# 公共基础模块（common）

## 功能概述
提供整个项目的公共基础设施，包括通用模型基类、统一异常、统一响应结构、序列化基类以及通用视图基类，所有业务模块都基于这些实现。

## 使用技术
- **Django ORM**：用于声明 `BaseModel`，提供字段、软删和审计功能。
- **Django REST Framework (DRF)**：实现 `BaseModelViewSet`、`BaseModelSerializers`，统一 API 行为。
- **Python 异常机制**：自定义业务异常类，统一错误返回。

## 关键代码文件及片段
### `common/models.py`
```python
class BaseModel(models.Model):
    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)
    is_deleted = models.BooleanField(default=False)

    class Meta:
        abstract = True

    def delete(self, using=None, keep_parents=False):
        # 软删实现
        self.is_deleted = True
        self.save()
```

### `common/response.py`
```python
class ApiResponse:
    @staticmethod
    def success(data=None, message="成功"):
        return Response({"code": 0, "msg": message, "data": data})

    @staticmethod
    def error(message="错误", code=1, data=None):
        return Response({"code": code, "msg": message, "data": data})
```

### `common/serialize.py`
```python
class BaseModelSerializers(serializers.ModelSerializer):
    def is_valid(self, *, raise_exception=False):
        valid = super().is_valid(raise_exception=raise_exception)
        if not valid and raise_exception:
            raise serializers.ValidationError(self._get_first_error_message(self.errors))
        return valid
```

### `common/views.py`
```python
class BaseModelViewSet(viewsets.ModelViewSet):
    # 统一的软删、权限、成员范围过滤实现
    def perform_destroy(self, instance):
        instance.delete()
```

## 为什么使用这些技术
- **抽象基类**让所有业务模型自动拥有审计字段，避免重复代码。
- **DRF**提供成熟的序列化、视图、分页、权限体系，能够快速构建一致的 RESTful API。
- **统一异常**让前端只需要关注 `code`、`msg` 两个字段，提升调试效率。

## 常见错误与排查
1. **模型未继承 `BaseModel`**：导致缺少 `created_at` 等字段，检查模型声明是否 `class XXX(BaseModel):`。
2. **序列化器未继承 `BaseModelSerializers`**：自定义错误信息失效，确保 `class XXXSerializer(BaseModelSerializers):`。
3. **视图集未使用 `BaseModelViewSet`**：软删失效或权限检查缺失，检查 `class XXXViewSet(BaseModelViewSet):`。
4. **异常未被捕获**：API 返回 500，确认 `settings.py` 中 `REST_FRAMEWORK['EXCEPTION_HANDLER']` 已指向项目统一异常处理器。
